# Project-Aplikasi-Sistem-Analisis-Jaringan-Sosial
Project ini merupakan aplikasi Sistem Analisis Jaringan Sosial yang dirancang untuk mengelola data pengguna serta menganalisis interaksi sosial dalam bentuk hubungan pertemanan. Aplikasi ini dikembangkan sebagai studi kasus penerapan berbagai struktur data dalam sebuah sistem terintegrasi.
